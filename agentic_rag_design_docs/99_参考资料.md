## 4. 本设计书与参考资料的关系

本设计书中的以下关键选择与以上资料直接相关：

- 使用 LangGraph 作为 Agent 编排中枢
- 使用 LangChain 承担 RAG 与 MCP 工具生态入口
- 使用 PostgreSQL + pgvector 作为统一底座
- 使用 Parent-Child Chunk + Contextual Chunking
- 使用 Hybrid Retrieval + Rerank
- 使用显式 Supervisor + 子图结构，而不是无约束大 Agent
- 使用四层记忆设计
- 使用 interrupt / approval / resume 形成 HITL 流程
- 使用流式步骤展示与可观测 trace
